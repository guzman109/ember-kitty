# Adding Ember to kitty

Clone this repository:

```sh
git clone https://github.com/ember-theme/kitty.git
```

Merge the theme files into kitty's themes directory:

```sh
mkdir -p ~/.config/kitty/themes
cp -R kitty/themes/. ~/.config/kitty/themes/
```

Open kitty's theme picker:

```sh
kitty +kitten themes
```

Pick the Ember theme you want to use.
